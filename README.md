# Aryan REST API

A RESTful API built with Node.js, Express, and MongoDB.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- Railway (Deployment)

## Features
- User CRUD API
- MongoDB integration
- REST architecture
- Deployed backend

## API Endpoints

GET /api/users  
POST /api/users  
PUT /api/users/:id  
DELETE /api/users/:id
